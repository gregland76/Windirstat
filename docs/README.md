# WinDirStat PowerShell - Documentation

Ce dossier contient la documentation du projet WinDirStat PowerShell.

La documentation correspond a la version 1.9 du projet, incluant le nouveau treemap a blocs plus carres dans le panneau gauche.

## Contenu

- `help.md` : guide utilisateur (installation, utilisation, depannage).
- `help.html` : guide utilisateur en version HTML.
- `changelog.md` : historique des mises a jour en version Markdown.
- `changelog.html` : historique de modifications en version HTML.
- `generate-screenshots.ps1` : script de regeneration des captures de documentation.
- `screenshots/` : captures d'ecran de l'application.

## Historique des mises a jour

L'historique du projet est disponible en deux formats :

- `changelog.md` pour la consultation directe dans le depot.
- `changelog.html` pour une lecture plus confortable dans un navigateur.

## Ouverture rapide depuis l'application

Un bouton **Docs** est disponible dans l'interface.
Il ouvre automatiquement ce dossier de documentation.

## Captures d'ecran

Les captures actuelles sont disponibles dans `docs/screenshots/` avec ces noms :

- `01-interface-principale.png`
- `02-apres-scan.png`
- `03-menu-contextuel.png`
- `04-fenetre-about.png`

Le guide utilisateur fait reference a ces fichiers.
Elles peuvent etre regenerees avec `generate-screenshots.ps1`.
